## 🔧 故障排查表

| 症狀 | 原因 | 解決方案 |
|------|------|--------|
| E-1 回傳 `sheet_not_found` | 分頁名稱不符 | 檢查 Sheet 分頁名是否完全一致「發送人員清單」 |
| E-1 回傳 `error` | doGet 未部署 | 在 Apps Script 重新部署為 Web App |
| E-1 全是空值 | 分頁無資料 | 在 Sheet 「發送人員清單」填入 5 位收件人 |
| E-2 回傳 `ok:false` | 儲位清單分頁不存在 | 建立「儲位清單」分頁，填入 BA ×32 |
| E-3 無商品資料 | 商品清單無資料 | 在「商品清單」填入至少 1 個商品 |
| 寄信失敗 | Gmail API 未授權 | 重做【C】授權步驟，確認「以你的名義傳送電子郵件」已勾選 |
| 照片上傳失敗 | Drive API 未授權 | 重做【C】授權步驟，確認「查看、建立、刪除 Google Drive 檔案」已勾選 |
