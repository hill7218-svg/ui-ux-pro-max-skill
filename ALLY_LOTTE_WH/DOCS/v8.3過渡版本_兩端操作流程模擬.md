# 🔄 v8.3 過渡版本 — 兩端操作流程模擬

**版本特性**: 保留 v8.2 DOUBLE CHECK + 新增預印貼紙 QR Code 快速路線  
**目標**: 降低 PDA 操作步驟、支援「謹慎模式」和「快速模式」並行

---

## 📍 整體架構

```
樂天 (鶯歌端)          預印貼紙                  你 (瑞芳端)
  商品             187 張貼紙                   收貨驗收
  │                  │                          │
  ├─ 品項分類         ├─ SKU                    ├─ 掃 QR Code
  ├─ 生成 187 張貼紙  ├─ 品名 + 箱數            │  (快速路線)
  └─ 貼紙內容:        ├─ 來源儲位 (C05A)        │
     · SKU            ├─ 入庫單條碼             ├─ 或掃商品條碼
     · 品名           ├─ 上架儲位 (BK/BA)       │  (謹慎路線)
     · 箱數           ├─ 上架條碼               │
     · 來源儲位       └─ QR Code                └─ DOUBLE CHECK
     · 入庫單              ↓                      ↓
     · 入庫單條碼     PDA 讀取                  拍照 32 格
     · 上架儲位       (自動帶出)                  ↓
     · 上架條碼                                 上架貨品
```

       · 主操作手 (掃貼紙/商品條碼)
       · 副操作手 (核對品名/箱數、點擊確認)
       · 攝影手 (32 格拍照對應)
