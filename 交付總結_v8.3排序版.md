## ✨ 核心功能實現

### 1️⃣ 自動排序功能 ✅

**實現位置**: `ALLY_LOTTE_QR_PRINTER_v8.3_排序版.html`

**功能**:
```
使用者操作: 選擇車次 → 設定張數 → 點「產生貼紙」

系統動作:
1. 篩選該車次的所有板
2. 按 SKU (A-Z) + 件數 (小→大) 排序
3. 重新分配入庫單號 (20260622-001/002/...)
4. 生成排序後的貼紙
```

**排序演算法**:
```javascript
function sortPalletsBySKUandQty(pallets) {
  let sorted = JSON.parse(JSON.stringify(pallets));
  sorted.sort((a, b) => {
    let skuCmp = a.sku.localeCompare(b.sku);        // SKU 字母序
    if(skuCmp !== 0) return skuCmp;
    return a.qtyPcs - b.qtyPcs;                     // 件數小到大
  });
  sorted.forEach((p, idx) => {
    p.orderId = '20260622-' + String(idx+1).padStart(3,'0');
  });
  return sorted;
}
```
