## 🗂️ 欄位定義（CSV / TSV）

```csv
source_id,sku,source_loc,trip,board_num,dest_location
LOTTE-20260622-Air_con622_1-001,BK9275,C05A0194-0,Air-con6/22-1,001,
LOTTE-20260622-Air_con622_1-002,BK9275,C05A0214-0,Air-con6/22-1,002,
```

| 欄位 | 說明 | 是否已填 |
|------|------|:---:|
| `source_id` | 來源單號（貼紙上的條碼） | ✅ 已填 |
| `sku` | 品號（對照用） | ✅ 已填 |
| `source_loc` | 鶯歌來源儲位（C05A 格式，僅供對照） | ✅ 已填 |
| `trip` | 行次（Air-con / regular） | ✅ 已填 |
| `board_num` | 板號 001–187 | ✅ 已填 |
| **`dest_location`** | **瑞芳目的儲位（BK/BA 格式）** | ⬅️ **請你填這欄** |

---

## ✍️ 填寫方式（CSV 範例）

只要把最後一欄 `dest_location` 填上瑞芳倉儲位：

```csv
source_id,sku,source_loc,trip,board_num,dest_location
LOTTE-20260622-Air_con622_1-001,BK9275,C05A0194-0,Air-con6/22-1,001,BK012-A01
LOTTE-20260622-Air_con622_1-002,BK9275,C05A0214-0,Air-con6/22-1,002,BK012-A02
LOTTE-20260622-Air_con622_1-003,BK9275,C05A0234-0,Air-con6/22-1,003,BK012-B01
```

### 📌 目的儲位格式（請確認）
我目前假設格式為 `BK###-A##`（如 `BK012-A01`）。
請告訴我瑞芳倉實際的編號規則，例如：
- `BK012`（純儲位號）
- `BK012-A01`（儲位+層架）
- `BA034/01`（斜線分隔）
