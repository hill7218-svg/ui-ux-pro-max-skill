📋 部署步驟

  1️⃣  列印器（辦公室）
      → 在 Windows/Mac 瀏覽器開 ALLY_LOTTE_QR_PRINTER_v6.9.html
      → 填入 Google Sheet ID
      → 讀入儲位清單 BA~BP
      → 產生貼紙（192 張 = 32 板 × 1 走道）
      → 列印（10cm × 10cm）

  2️⃣  移出端 APP（鶯歌 PDA）
      → PDA WebView 加載 ALLY_LOTTE_WAREHOUSE_v6.9.html
      → 開工設定 (S6 選車號/走道/趟次)
      → S1 掃商品 EAN → 掃來源 QR → 掃儲位 Code128
      → S3 封板（若被佔可改）→ 確認
      → S4 關單 → 入庫單上傳 Sheet

  3️⃣  收貨端 APP（瑞芳 PDA）
      → 同一 HTML（或另一台 PDA）
      → S0 選「D. 卸貨驗收」→ S8
      → 掃棧板 QR（查品項 + 讀最終儲位）
      → 掃儲位 Code128（軟驗證）
      → 黃燈時輸入覆寫原因 → 確認上架
